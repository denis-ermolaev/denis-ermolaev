# AGENTS

## Цель
Этот репозиторий содержит персональный README и иконки для представления стекa технологий. Основная задача — поддерживать README в аккуратном и консистентном виде.

## Правила работы для задачи "синхронизировать README с README-DEV и скачать SVG"
1. Сначала изучи текущий README-DEV и связанные файлы в папке assets/icons.
2. По запросу пользователя тебе нужно обновить `README.md`, а именно привести его в тот же вид, что и `README-DEV.md`, но скачать все svg файлы по ссылкам(например, `https://img.shields.io/badge/Tools-Git%20%7C%20Docker%20%7C%20Linux%20%7C%20PyTest%20%7C%20Ruff%20%7C%20ty%20%7C%20uv-4CAF50?style=flat-square`), поместив их в assets/icons в соотвествующую папку, и после в `README.md` ссылаться на них из локальных файлов, как на svg изображения.

## Структура репозитория
- README.md — основной файл профиля.
- README-DEV.md — версия для разработки/черновик.
- assets/icons/ — общие иконки.
- assets/icons/stack/ — SVG-бейджи для стека технологий.
